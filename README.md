# Robustness for random forests on multiclass data

## Installing

- Using poetry:

  - Install poetry:

  ```bash
  curl -sSL https://install.python-poetry.org | python3 -
  ```

  - Install dependecies:

  ```bash
  poetry install
  ```

- Using venv:

```bash
# Create a virtual environment named "venv"
python3 -m venv venv

# Activate it
# macOS / Linux
source venv/bin/activate
# Windows (PowerShell)
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```
